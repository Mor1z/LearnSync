Курс SQL с нуля

SQL для начинающих: Основы работы с базами данных с нуля

Краткое описание:

Этот курс создан для тех, кто хочет научиться работать с базами данных: извлекать, изменять, добавлять и удалять данные. SQL (Structured Query Language) - это стандартный язык для управления реляционными базами данных. Если вы новичок в работе с данными или только начинаете свой путь в IT и хотите освоить фундаментальный навык, этот курс - идеальная отправная точка.

Кому подходит этот курс:

  • Аналитикам данных и тем, кто работает с большими объемами информации.
  • Разработчикам (как фронтенд, так и бэкенд), которым необходимо взаимодействовать с базами данных.
  • Тестировщикам, для проверки данных в базах.
  • Всем, кто хочет освоить базовые принципы работы с реляционными базами данных.

Что вы узнаете:

  • Основы баз данных: Что такое реляционные базы данных и зачем нужен SQL.
  • Получение данных: Как извлекать данные из одной или нескольких таблиц с помощью оператора `SELECT`. 
    
SELECT column1, column2
FROM table_name
WHERE condition;
  • Фильтрация данных: Использование `WHERE` для отбора строк по заданным условиям. 
    
SELECT *
FROM Customers
WHERE Country = 'Mexico';
  • Сортировка данных: Как упорядочивать результаты с помощью `ORDER BY`. 
    
SELECT *
FROM Products
ORDER BY Price DESC; /* Сортировка по убыванию цены */
  • Ограничение выборки: Использование `LIMIT` или `TOP` (зависит от СУБД) для ограничения количества возвращаемых строк. 
    
SELECT *
FROM Orders
LIMIT 10; /* В MySQL и PostgreSQL */
  • Агрегатные функции: `COUNT`, `SUM`, `AVG`, `MIN`, `MAX` для выполнения вычислений над наборами данных. 
    
SELECT COUNT(CustomerID)
FROM Customers;

SELECT AVG(Price)
FROM Products;
  • Группировка данных: Использование `GROUP BY` для группировки строк по значениям столбцов. 
    
SELECT Country, COUNT(CustomerID)
FROM Customers
GROUP BY Country;
  • Объединение таблиц (JOINs): Как объединять данные из нескольких таблиц (INNER JOIN, LEFT JOIN, RIGHT JOIN, FULL OUTER JOIN). 
    
SELECT Orders.OrderID, Customers.CustomerName
FROM Orders
INNER JOIN Customers ON Orders.CustomerID = Customers.CustomerID;
  • Добавление данных: Использование оператора `INSERT INTO` для добавления новых строк. 
    
INSERT INTO Customers (CustomerName, City, Country)
VALUES ('Cardinal', 'Stavanger', 'Norway');
  • Изменение данных: Использование оператора `UPDATE` для изменения существующих строк. 
    
UPDATE Customers
SET City = 'Oslo'
WHERE CustomerID = 1;
  • Удаление данных: Использование оператора `DELETE FROM` для удаления строк. 
    
DELETE FROM Customers
WHERE CustomerName = 'Cardinal';
  • Создание и изменение таблиц (базово): Операторы `CREATE TABLE` и `ALTER TABLE`. 
    
CREATE TABLE Persons (
    PersonID int,
    LastName varchar(255),
    FirstName varchar(255)
);

Требования:

  • Желание учиться и практиковаться.
  • Компьютер и доступ к какой-либо СУБД (системе управления базами данных, например, SQLite, MySQL, PostgreSQL) или онлайн-песочнице SQL.

Почему стоит пройти этот курс:

  • Фундаментальный навык: SQL является одним из основных языков для работы с данными.
  • Широкое применение: SQL используется во многих областях IT и за ее пределами.
  • Практический подход: Отработаем основные запросы на реальных примерах данных.
  • Основа для дальнейшего роста: Знание SQL откроет двери к изучению баз данных более глубоко и работе с инструментами анализа данных.

  • Добавлять, изменять и удалять данные в базе.
  • Понимать основы работы с реляционными базами данных.
